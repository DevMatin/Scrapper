# Scrapper SEO config (`~/.config/scrapper-seo/`)

Created automatically by `scrapling seo` commands.

| File | Purpose |
|------|---------|
| `google-api.json` | Google API keys / OAuth references |
| `backlinks-api.json` | Moz, Bing backlink API keys |
| `dataforseo-costs.json` | DataForSEO budget tracking |
| `drift/` | SQLite baselines for SEO drift monitoring |

Repo `.env` provides `SUPABASE_*`, `GOOGLE_MAPS_API_KEY`, and optional `GOOGLE_API_KEY`.
